Getting Started with the API

Marvel's API gives developers acccess to Marvel's database on comics, characters, creators, and more.

Getting Access to the API

First you will need to gain access to the API itself. You will need to have created an account on Marvel's site. I have included the link to the page below.

Get Access to the API

Setting up your API

Once you have signed into or created your Marvel Account you should be taken to a page like the one pictured above. Bookmark this page as it will be important. Once there you will be given both a public and private API key. You will also be able to choose the domains that will be able to access your new API keys.
